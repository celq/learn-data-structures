整型指针变量p，存储的值是整型变量age的内存地址。  
符号`&`是取地址符，那么&age = 000000000062FE1C;由于指针变量本身也是一个变量，所以p也有一个自己的地址，对其取地址操作&p=000000000062FE10;
而`*p`则是获取指针变量p存储的值对应内存地址中存储的值。
